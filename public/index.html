HerumHai

A real-time, serverless Stremio / Nuvio addon that resolves direct HLS & MP4 streams with anti-decoy filtering and proxy headers.

⚡ Install in Stremio View Manifest
loading…
🛡️

Anti-Decoy Filter

Automatically discards tutorial, sample, trailer, and any MP4 under 50MB. Real streams only.

⚡

Real-Time Resolution

Spawns headless Chromium on each request, navigates embeds, and captures HLS / DASH / MP4 streams.

🔑

Proxy Headers

Captures User-Agent, Referer, and Cookie from the live session and injects them via behaviorHints.

☁️

Vercel Serverless

Built on @sparticuz/chromium + puppeteer-core for AWS Lambda compatibility. No cold-start infrastructure.
