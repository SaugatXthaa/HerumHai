ℹ️ How It Works — "Double-Click" Method

First Click (Scrape): When you click a movie for the first time, the addon triggers a background scrape of all 38+ sources. Instead of returning an empty array, it shows a placeholder card: "⚙️ Scraping 38+ Sources... Close & Reopen in 10 Seconds!"


Second Click (Play): Close the stream view, wait 10 seconds (while the background scraper finishes), then click the movie again. The cache is now populated → all 38+ real streams appear instantly (<50ms).


Pure HTTP: No Puppeteer, no Playwright, no FlareSolverr. All sources are scraped via axios with randomized browser headers (User-Agent, Accept-Language, Referer rotated per request).


URL Validation: Every stream URL is validated — only 200/206 with video content-type pass. This eliminates all MPV "unrecognized file format" errors.


Memory Footprint: ~30MB total (pure HTTP = ~0MB per request). 20 concurrent requests use less memory than a single browser tab. Perfect for 512MB SnapDeploy free tier.
